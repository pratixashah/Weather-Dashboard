Summary: Latitude vs. X

This project was aimed to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, I have use a Python library, the OpenWeatherMap API.

Once data received from API, I have used Matplotlib to plot various aspects of the weather vs. latitude. It includes temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanation and descriptions of any trends and correlations witnessed.
